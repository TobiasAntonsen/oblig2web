Informasjon om noen relevante emner

Math Ml er et matematisk markup language. Brukes til å beskrive og integrere matematiske formeler til WWW sider og andre dokumenter.
For mer informasjon trykk deg videre i navigasjonen over.

GitHub er et vector bilde format for 2D grafikk. Det står for Scalable Vector Graphics og støtter interaktivitet og animasjon i dokumenter og programmer.
For mer informasjon trykk deg videre i navigasjonen over.

SVG er en type version control repository, også kalt Git, og fungerer som en hosting tjeneste som er web-basert, og er populært brukt for kode.
For mer informasjon trykk deg videre i navigasjonen over.
